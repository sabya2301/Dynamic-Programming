25) Binary Strings 

Define two DP arrays dp0[] and dp1[]. dp1[] stores binary strings ending with '1' and dp0[] stores binary strings ending with '0'. Now, dp0[0] and dp1[0] are 0 and
dp0[1] and dp1[1] are 1(since only one value is there). For dp1[2], the values can be '01' or '11' (since ending with 1), so it takes value from both dp1[1] ('0') 
and dp1[1] ('1'). In the similar way the rest of the array is formed. 
NOTE : This approach can be further optimised by not declaring an array and storing the count in two variables(as we dont require the values from previous cells).
